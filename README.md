A machine learning project that predicts startup profit based on key investment factors such as R&D spend, administration, and marketing expenditure.

This project compares multiple regression models — Linear Regression, Decision Tree, and Random Forest — to evaluate performance using standard metrics like MAE, MSE, RMSE, and R² score. It identifies the best-performing model for accurate profit prediction.

Built using Python, Pandas, NumPy, and Scikit-learn, this project demonstrates end-to-end workflow including data preprocessing, model training, evaluation, and comparison.
